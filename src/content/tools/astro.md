Astro 是一个现代静态网站生成器，特别适合构建内容驱动的网站，如博客、文档站、企业官网等。

## 核心特点

### 1. 内容优先

Astro 的设计理念是"让内容成为第一公民"：

- 内置 Markdown 和 MDX 支持
- Content Collections 提供类型安全的内容管理
- 零 JavaScript 默认输出

### 2. 性能卓越

- 默认零 JavaScript 输出
- 按需加载组件
- 自动图片优化
- 内置代码分割

### 3. 灵活组件

支持多种 UI 框架的组件：

- React
- Vue
- Svelte
- SolidJS
- 纯 Web Components

### 4. 开发体验

- 热更新（HMR）
- TypeScript 原生支持
- 丰富的集成生态

## 适用场景

- 个人博客
- 文档网站
- 企业官网
- 作品集
- 知识库

## 为什么选择 Astro

相比 Next.js 和 Nuxt，Astro 更轻量、更专注。如果你主要做内容型网站，Astro 是更好的选择。
