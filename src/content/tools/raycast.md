Raycast 是 macOS 上的高效启动器，可以替代系统的 Spotlight，大幅提升工作效率。

## 核心功能

### 1. 快速启动

- 应用程序
- 文件夹
- 书签
- 最近使用项目

### 2. 剪贴板历史

- 自动保存剪贴板历史
- 支持图片和代码片段
- 快速粘贴历史内容

### 3. 窗口管理

- 快速调整窗口大小
- 屏幕分屏
- 窗口布局预设

### 4. 自定义脚本

- 运行自定义脚本
- 集成各种服务
- 扩展功能无限

## 为什么推荐

1. **速度更快**：比 Spotlight 响应更迅速
2. **功能更强**：丰富的扩展功能
3. **可定制**：支持自定义脚本和工作流
4. **美观**：界面设计精美

## 使用技巧

- 熟练使用快捷键
- 利用扩展功能
- 自定义常用工作流
- 配合剪贴板历史使用
